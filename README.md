# KOSMOS - Магазин товаров оптом и в розницу

Веб-приложение для продажи товаров с поддержкой оптовых и розничных цен. Полностью функциональное, не требует серверной части.

## ✨ Особенности

- 📦 **6 категорий товаров** с подробными описаниями
- 🖼️ **Фотографии товаров** для каждого артикула
- 💰 **Двойное ценообразование** - розница и опт
- 🛒 **Умная корзина** с автосохранением в localStorage
- 📱 **Адаптивный дизайн** - работает на телефонах и ПК
- 🔐 **Защита от XSS** - все данные экранированы
- 💬 **Интеграция с WhatsApp** - отправка заказов напрямую
- 📲 **Offline режим** - работает без интернета благодаря Service Worker
- ⚡ **Без зависимостей** - чистый JavaScript, HTML5, CSS3

## 🛠️ Настройка

### Изменить номер владельца WhatsApp

Отредактируйте `index.html` и найдите строку:
```javascript
const OWNER_PHONE = '996999123456'; // Замените на ваш номер
```

Номер должен быть в формате: страна-код + номер без пробелов и символов.
- **Кыргызстан**: `996xxxxxxxxx`
- **Россия**: `7xxxxxxxxxx`

### Изменить товары и цены

Отредактируйте `index.html`, найдите объект `const products = {` и добавьте/измените товары:

```javascript
const products = {
    1: [ // Категория 1
        {
            id: 1,
            name: 'Название товара',
            description: 'Описание',
            priceRetail: 100,      // Розничная цена
            priceWholesale: 85,    // Оптовая цена
            wholesaleFrom: 10,     // От какого кол-ва опт цена
            image: 'FILENAME.PNG'  // Имя файла в assets/images/
        },
        // ... другие товары
    ]
};
```

### Добавить свои картинки товаров

1. Поместите картинку в папку `assets/images/`
2. Укажите имя файла в свойстве `image` товара
3. Используйте формат PNG или JPG

Приложение имеет встроенную защиту от:
- ✅ XSS атак (escapeHtml, textContent вместо innerHTML)
- ✅ Переполнения localStorage (проверка размера)

## 📊 Функции

### Для покупателя
- Просмотр товаров по категориям
- Добавление товаров в корзину
- Изменение количества товаров
- Автоматический расчет цены (розница/опт)
- Оформление заказа
- Отправка заказа в WhatsApp владельцу

### Для владельца
- Товары остаются в памяти браузера покупателя
- История заказов в localStorage
- Получение заказов в WhatsApp с полной информацией
- Нет затрат на сервер и БД
